# Lua Potrace

Lua potrace is a port of [PotraceJS](https://github.com/kilobtye/potrace) which is a port of [Potrace](http://potrace.sourceforge.net/).

Using [Bitmap](https://github.com/max1220/lua-bitmap) for bitmap decode.
